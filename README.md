## :rocket: Quick Start

```bash
# Clone
git clone https://github.com/coder-nishanth/codah-music.git
cd codah-music

# Install & Run
flutter pub get
flutter run
```

### Build Release

```bash
flutter build windows --release
```

Output: `build/windows/x64/runner/Release/codah-music.exe`
