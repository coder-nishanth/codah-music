## Features

- **YouTube Music** — Search and stream any song
- **Lyrics** — Synced lyrics with LRCLib & Lyrica
- **Charts** — Billboard Hot 100, Billboard 200, Spotify charts
- **Local Library** — Play your local audio files
- **Auto Updates** — Built-in update notifications
- **Material 3** — Dynamic theming with blur effects

---

## Quick Start

```bash
git clone https://github.com/coder-nishanth/codah-music.git
cd codah-music
flutter pub get
flutter run
```

Build release: `flutter build windows --release`
